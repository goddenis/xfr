Сборка приложения производится чпри помоши команды gtadlew jar
Для создания таблиц выполнить schema.sql
Для запуска можно использовать файлы start.bat или start.sh 
При запусек можно испорльзовать три параметра :
-i=C:\input    -папка с входящими файлами
-o=C:\output   -папка с исходящими файлами
-e=C:\error    -папка сфайлами не прошедшими парсинго
-t=1000        -период в с секундами
если приложение не найдет дириктории они будут созданы тпакже будет создана директория log куда будут выводиться логи исполнения
